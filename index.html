Functions Exercises:

  1. Define a function named "isTrue" that takes in an input argument and returns true if the provided value is the boolean value of true.
  2. Define a function named "isFalse" that takes in an input argument and returns true if that value provided is the boolean value of false.
  3. Define a function named "not" that takes in an input argument and returns the boolean opposite of the provided value
  4. Define a function named "addOne" that takes in an input argument. If the provided value is a number or a string containing a number, "addOne" should return the number plus one. Non-numeric inputs provided should return NaN
  5. Define a function named "isEven" takes in an input argument and returns true if the input is an even number or string containing an even number. "isEven" should return false when provided any other inputs
  6. Define a function named "isIdentical" that takes in two input arguments. The isIdentical function should return true if both input values are equal in value and equal in data-type.
  7. Define a function named isEqual that takes in two input arguments. The isEqual function should return true if both inputs are equal in value only, ignoring data-type
  8. Define a function named "or" that takes in two input arguments. The "or" function should return the result of a boolean "or" operation on both inputs.
  9. Define a function named "and" that takes in two inputs and returns the result of a boolean "and" operation
  10. Define a function named "concat" that takes in two inputs. Concat returns the result of string concatenation of both arguments. Numbers provided to concat should be treated as strings.
